Instalação

Linux
# apt-get install git

Mac OS
$ brew install git

Windows
desktop.github.com
*instruções abaixo para executar o shell*

$ git --help

Lista os subcomandos e descreve alguns conceitos.

$ git config --global user.name "Marina Limeira"

Configura o nome do usuário que fará os commits.

$ git config --global user.email marinaflessa@gmail.com

Configura o e-mail do usuário que fará os commits.

$ git init

Inicia um repositório.

$ git remote add origin repo

Conecta a um repositório remoto.

$ git clone url

Cria uma cópia do repositório localmente.

https x ssh

$ git add file

Prepara os arquivos para commit.

$ git rm file

Remove o arquivo do repositório.

$ git commit -m message

Commita os arquivos que foram alterados.

$ git push origin branch

Envia os commits para um branch remoto.

$ git pull origin branch

Atualiza os commits de um branch remoto.

$ git diff

Mostra as alterações locais.

$ git log

Mostra os commits que foram feitos.
